Проект выполняется в ходе курсовой работы. 
Проект создавался группой из 3 человек. 
Price_Tracker - это сайт, который анализирует цену интересного вам товара. 
В ходе проекта реализованы парсеры и ML, для хранения данных подключена база данных SQL.
Структура сайта реализована с помощью шаблона django.
